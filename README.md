# BestTools - منصة أفضل الأدوات الرقمية

موقع直观 وعصري لعرض ومراجعة أفضل الأدوات الرقمية (AI, Design, Productivity, Marketing, Development).

## ✅ المميزات

### المتجاوبة
- **هواتف**: من 320px فما فوق
- **تابلت**: من 768px إلى 1024px
- **حواسيب**: من 1025px فما فوق
- **تلفاز ذكي**: من 1921px فما فوق (4K, 8K)

### التقنيات
- **HTML5** مع دعم كامل لـ Semantic markup
- **CSS3** مع Flexbox و Grid والمتغيرات
- **JavaScript** نقي (Vanilla JS) بدون frameworks
- **Dark/Light Mode** تبديل تلقائي
- **Lazy Loading** للصور
- **PWA** (Progressive Web App)

### الأداء
- **أحجام ملفات صغيرة**: CSS ~38KB, JS ~53KB
- **تحميل سريع**: < 2 ثانية على 3G
- **Scroll animations** سلسة
- **GPU accelerated** animations

## 🎯 الصفحات

1. **index.html** - الرئيسية مع Hero و featured tools
2. **tools.html** - قائمة كاملة بفلترة حسب الفئة
3. **tool-detail.html** - صفحة تفاصيل ديناميكية لكل أداة
4. **top10.html** - أفضل 10 أدوات مرتبة
5. **about.html** - عن الموقع
6. **contact.html** - نموذج تواصل

## 🛠️ التخصيص

### تغيير الألوان
عدل في `styles.css` في قسم `:root`

### إضافة أدوات جديدة
أضف في `main.js` في مصفوفة `toolsData`

### تغيير اللغة
الموقع يدعم RTL للعربية.
